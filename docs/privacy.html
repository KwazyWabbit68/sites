Privacy Policy for SpeedGuard

Effective Date: 03/19/2023

Developer Information and Privacy Point of Contact

Kurt Vermote is the developer of SpeedGuard. If you have any questions or concerns about this Privacy Policy or the app's data handling practices, please contact us at privacy@speedguard.app.

Personal and Sensitive User Data

No data is shared with third parties.

SpeedGuard accesses and collects the following types of personal user data:

  • The mobile device's geo location in order to calculate the current and average speed and marking any custom check zones while the app is running, be it in the foreground or in the background.
  • The user preferences that are set within the app, such as language, units of measurement and speed limits.
  • Anonymous crash and diagnostics information may be collected for product improvement.

Data Handling Procedures

We take all reasonable measures to ensure the security and confidentiality of the personal user data we collect. We use this data only to provide the services offered by SpeedGuard, no personal user data is shared with third parties.
Moreover, any data that may be stored off-device will be anonymous and can therefore not be traced back to the user or his/her device.

Data Retention and Deletion Policy

The begin and end locations of user created check zones may be temporarily stored off-device, but always anonymously.
It is not possible for us to trace this info back to the user that created the check zone or his/her device. As such, no personal user data is retained by the application, so there is indeed no account info to delete.
